AnimationsAreEasy

Коллекция из 14 CSS-микроанимаций — без JavaScript

2025 Pet-проект

Ситуация

В работе над разными проектами разработчики постоянно говорили, что им нужны Lottie JSON-анимации даже для простейших вещей: hover-эффект на кнопке, спиннер загрузки, плавное появление блока. Я смотрел на эти задачи и не понимал — зачем тащить тяжёлый Lottie-плеер и внешние файлы ради того, что можно сделать на чистом CSS?

В какой-то момент я решил проверить: а насколько далеко можно зайти на одном CSS? Без библиотек, без JS-анимаций, без Lottie. Взял и собрал 14 микроанимаций — от спиннера до 3D-карточки. Результат меня убедил: если анимация не сложная, CSS справляется на ура. Никакие Lottie не нужны.

Задача

Разработать библиотеку CSS-микроанимаций, которая:

  • Работает без JavaScript — только CSS transitions, transforms и keyframes.
  • Содержит разнообразные эффекты: от skeleton-загрузки до 3D-карточек и спиралей.
  • Совместима с любым стеком — React, Vue, Angular или просто HTML.
  • Сопровождается демо-страницей для быстрого просмотра и выбора.

Действия

1

Исследование и отбор эффектов

Я проанализировал, какие микроанимации чаще всего встречаются в современных интерфейсах, и отобрал 14 эффектов, покрывающих основные потребности:

  • Обратная связь: Shake/Nudge, Glow/Pulse Button, Bounce/Elastic.
  • Загрузка: Skeleton Shimmer, Loading Spinner, Pulse/Ripple.
  • Навигация: Cursor Drag, Card Deck.
  • Декоративные: Emoji Spiral, Floating/Bobbing, Gradient Flow, Swipe Reveal.
  • Текст: Typing Effect.
2

Разработка анимаций

Каждая анимация реализована исключительно средствами CSS:

  • @keyframes — для сложных последовательных анимаций (card deck, emoji spiral).
  • transition — для плавных интерактивных состояний (glow button, card flip).
  • transform — для производительных перемещений и поворотов (bounce, floating).
  • GPU-композиция — только свойства opacity и transform для 60fps.
3

Демо-страница и деплой

Создал демо-страницу с сеткой превью, где каждая анимация запускается по ховеру или фокусу. На мобильных устройствах (где нет hover) используется IntersectionObserver — анимация срабатывает при появлении в области видимости.

Страница опубликована на GitHub Pages, иконки — Phosphor (MIT). Репозиторий открыт для использования и форков.

Результат

  • 14 анимаций — от skeleton-загрузки до 3D-карточек и спиралей.
  • Zero JS-зависимостей — библиотека подключается одним CSS-файлом.
  • Доступность — все анимации работают по hover, focus и (на мобильных) через IntersectionObserver.
  • MIT-лицензия — можно использовать в коммерческих проектах.
